Replace Temporary operands with the corresponding allocated physical register (Register) OR memory location (Offset).

prepare() → None[source]

Perform all preparatory steps related to smart register allocation:

  • Dataflow analysis to compute the liveness range of each temporary.

  • Interference graph construction.

  • Graph coloring.

  • Associating temporaries with actual locations.

build_interference_graph() → None[source]

Build the interference graph (in self._igraph). Vertices of the graph are temporaries, and an edge exists between temporaries iff they are in conflict.

smart_alloc() → None[source]

Allocates all temporaries via graph coloring. Prints the colored graph if self._debug_graphs is True.

Precondition: the interference graph _igraph must have been built.
