RegAlloc.SequentializeMoves module

CAP, SSA Intro, Elimination and Optimisations Helper functions to convert a CFG out of SSA Form for the Smart Allocator.

RegAlloc.SequentializeMoves.generate_smart_move(dest: DataLocation, src: DataLocation) → List[Instru3A | Comment][source]

Generate a list of move, store and load instructions, depending on whether the operands are registers or memory locations. This is an helper function for sequentialize_moves.

RegAlloc.SequentializeMoves.generate_xor_move(r1: Register, r2: Register) → List[Instru3A | Comment][source]

Generate a list of xor instructions to swap the two registers, without use of a temporary location. This is an helper function for sequentialize_moves.

RegAlloc.SequentializeMoves.sequentialize_moves(parallel_moves: Set[Tuple[DataLocation, DataLocation]]) → List[Instru3A | Comment][source]

Take a set of parallel moves represented as (destination, source) pairs, and return a list of sequential moves which respect the cycles. Use the register tmp S2 for the cycles. Return a corresponding list of RiscV instructions. This is an helper function called during SSA exit.
