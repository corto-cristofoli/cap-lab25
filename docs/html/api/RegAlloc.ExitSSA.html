RegAlloc.ExitSSA.generate_moves_from_phis(phis: List[PhiNode], parent: Block) → List[Instru3A | Comment][source]

generate_moves_from_phis(phis, parent) builds a list of move instructions to be inserted in a new block between parent and the block with phi nodes phis.

This is an helper function called during SSA exit.

RegAlloc.ExitSSA.exit_ssa(cfg: CFG, is_smart: bool) → None[source]

exit_ssa(cfg) replaces phi nodes with move instructions to exit SSA form.

is_smart is set to true when smart register allocation is enabled (Lab 5b).
